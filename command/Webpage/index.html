Controlls

Arrow Keys: Control cursor

Enter Key: Sends rover to location of cursor

Each square is 30cm by 30cm

Space Key: Cancel autonomous driving

Driving Modes

Simple: Rover will reach target coordinates via simplest path

Full Discovery: For each square rover will scan all adjacent blocks for obstacles

Destination Discovery: Rover will do Simple mode until it reaches target coordinates where it will scan all adjacent blocks for obstacles

Fully Autonomous Explore: Rover will explore the full map with no extra controls from user, location of cursor does not matter for this mode
